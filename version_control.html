Version Control | Git & GitHub

Version Control with Git. Each development team uses a version management method to document and control the changes made to their code over time. In the web and open source world, two common systems for this are SVN and GIT. We'll learn and practice these methods while focusing on Git, it's a relatively untrivial and complicated version management system, but when you get to know it in depth you understand why one of the world's leading companies is Github - a cloud service that provides Git services to organizations like jQuery, Twitter, Facebook and Adobe.
