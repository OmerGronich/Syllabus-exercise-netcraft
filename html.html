HTML5 web page

The building block of the world's most popular development platform! HTML is a simple but full of capabilities and complexity that every developer must know. It is used as the basic building block for any webpage, as well as many applications across a variety of platforms (such as mobile). We will learn how to create Web page and Web application structures, correct semantics and Markup patterns. In addition, we will review optimal web page architecture in terms of structure and best practices to use based on HTML5 Boilerplate, the popular and high-quality HTML skeleton.
