TypeScript

The TypeScript language is an extension for the JavaScript language that comes to complement functionality and semantics that do not exist in JavaScript and allows for complex and complex code development in a more structured way. Within the language you can define variable types and functions, which allow checking and detection of errors already in the writing and planning phase of the code.
