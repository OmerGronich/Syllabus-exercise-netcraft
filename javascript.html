Programming and development using: JavaScript

JavaScript structures for developing smart and dynamic interfaces. JavaScript is the "English" of the Internet, and the popular and widely used programming language in the world, by a considerable margin.

As a programming language, JavaScript is extremely versatile and diverse and has various paradigms that live harmoniously (relatively): Functional programming, Prototypal Inheritance, Object oriented programming and more. In the course we will learn the basic and advanced structures of the language so that we can develop smart and dynamic interfaces that have logic, presentation and data updating and communication with the server.
